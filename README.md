
# CSR Generation Tool

Generates Certificate Signing Request and Private Key

## Usage/Examples

```bash
./csrgen.py

CN:
Organization:
Organizational Unit:
Locality:
State:
Country:
SANs (ex. domain001.com,domain002.com) :
```
